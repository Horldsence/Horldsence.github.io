# Chiptune Lab

Compose classic chip melodies with a modern workflow. Edit wavetables, sequence patterns, and export clean renders.

## Features

- **Pattern tracker** with per-step automation
- **Wavetable editor** and sample import
- **Exports** to WAV, OGG, and NSF

## Installation

Download the latest release and follow the installer for your platform.
