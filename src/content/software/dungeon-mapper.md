# Dungeon Mapper

Lay out rooms, corridors, and props with a palette of pixel tiles. Export static maps or animated flythroughs for your players.

## Features

- **Tile palette** with snapping grid
- **Layers** for lighting, props, and secrets
- **Exports** to PNG, GIF, and Foundry-ready JSON

Grab the latest build for your OS from the releases page.
