# Pixel Notes

Take quick snippets without leaving your keyboard. Pixel Notes stores everything locally first, then syncs when you are back online.

## Features

- **Command Palette** for all actions
- **Local-first sync** with conflict resolution
- **Theming** with custom palettes

## Installation

Download the latest build from the releases page and run the installer.
